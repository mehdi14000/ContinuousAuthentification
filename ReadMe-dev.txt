IMPORTANT:

Pour pouvoir faire tourner le programme sur eclipse il faut ajouter la librairie gson-2.3.1.jar au project
Pour cela sur Eclipse on ira dans Project -> 
				  	    Properties -> 
						Java Build Path -> 
							Onglet Librairies -> 
								add External JARs 
								 	selectionner gson-2.3.1.jar et OK

Puis Run as a Java Application.
